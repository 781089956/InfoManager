# InfoManager
## 特征
* 支持多条件表达式的与或运算
* 支持类似于sql的语言操作习惯
* 支持脚本指令
## 如何使用
1. 运行程序
2. 键入help可获取帮助信息
3. 输入 `open 文件名` 以打开对应的csv文件
4. 通过 `select update delete insert` 进行操作查询
5. 输入 `save 文件名` 以保存数据 
6. 输入 `exit` 正常退出程序
## 依赖
使用了GNU的regex正则库，由于该库使用基于32位编译，因此程序暂时也只能支持32位。
